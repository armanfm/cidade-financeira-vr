# Cidade Financeira VR

Projeto desenvolvido em Unity com integração Meta XR SDK para criação de um ambiente virtual interativo em realidade virtual.

## Tecnologias Utilizadas

- Unity 6 LTS
- Meta XR All-in-One SDK
- OpenXR
- C#
- Assets 3D Low Poly

## Funcionalidades

- Ambiente urbano 3D
- Posto de combustível interativo
- Veículo 3D com física
- Estrutura preparada para VR
- XR Origin configurado
- Colisões e física básica

## Estrutura do Projeto

- Assets
- Packages
- ProjectSettings

## Observações

O projeto foi desenvolvido utilizando Meta XR SDK em Unity 6.  
Alguns componentes XR apresentam incompatibilidades conhecidas devido à versão beta do simulador.

## Execução

1. Abrir o projeto na Unity 6 LTS
2. Aguardar compilação dos pacotes
3. Abrir a cena principal
4. Executar no modo Play
